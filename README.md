Nama Proyek: Form-Kehadiran
Aplikasi berbasis desktop ini berfungsi sebagai alat praktis untuk mencatat dan mengelola data kehadiran peserta acara, khususnya webinar. Program ini memungkinkan pengguna untuk memasukkan data kehadiran secara efisien, memantau status kehadiran, dan menyimpan informasi peserta.
Fitur Utama dan Wawasan Praktis:
  1.	Pencatatan Data Peserta: Pengguna dapat dengan cepat memasukkan nama dan asal peserta (instansi) untuk mencatat kehadiran.
  2.	Pemantauan Status Kehadiran: Program menampilkan status kehadiran (misalnya, "Hadir") untuk setiap peserta yang terdaftar, memberikan informasi langsung mengenai partisipasi.
  3.	Pengelolaan Data Tepat Guna:
    •	Fungsi Tambah memungkinkan penambahan data peserta baru ke dalam tabel utama.
    •	Fungsi Hapus menyediakan cara mudah untuk menghapus data peserta yang tidak relevan atau salah input.
  4.	Basis Data Terintegrasi: Program terhubung dengan basis data MySQL, menggunakan driver mysql-connector-j-8.0.11.jar, memastikan data kehadiran tersimpan secara persisten dan aman.
  5.	Antarmuka Pengguna Sederhana (GUI): Dibangun menggunakan NetBeans dan library Swing, program ini memiliki antarmuka yang intuitif. Hal ini memudahkan operator atau admin dalam memasukkan dan melihat data tanpa kebingungan.
Teknologi yang Digunakan:
  •	Bahasa Pemrograman: Java
  •	IDE: Apache NetBeans
  •	Antarmuka: Swing (Java GUI Toolkit)
  •	Basis Data: MySQL
  •	Konektor: mysql-connector-j-8.0.11.jar
Cara Kerja Praktis:
  Program ini mengatasi tantangan pencatatan kehadiran manual. Operator cukup mengisi form dengan Nama Peserta dan Asal Instansi. Setelah menekan tombol Tambah, data langsung tersimpan ke basis data. Data kehadiran yang tersimpan dapat segera diekspor atau diolah lebih lanjut untuk kebutuhan laporan atau sertifikat.
