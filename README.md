# Aplikasi Form-Kehadiran

Aplikasi desktop ini berfungsi untuk mencatat dan mengelola data kehadiran peserta acara, terutama webinar. Program ini memungkinkan input cepat Nama dan Asal Instansi peserta, serta menampilkan status kehadiran. Program ini mengatasi tantangan pencatatan kehadiran manual. Operator cukup mengisi form dengan Nama Peserta dan Asal Instansi. Setelah menekan tombol Tambah, data langsung tersimpan ke basis data. Data kehadiran yang tersimpan dapat segera diekspor atau diolah lebih lanjut untuk kebutuhan laporan atau sertifikat.

## Fitur Utama

* **Pencatatan Kehadiran:** Input data peserta (Nama, Asal Instansi) secara efisien.
* **Pengelolaan Data:** Fungsi Tambah dan Hapus untuk memperbarui data dalam tabel secara langsung.
* **Integrasi Basis Data:** Penyimpanan data persisten dan aman menggunakan **MySQL** (`mysql-connector-j`).
* **Antarmuka Pengguna (GUI):** Dibangun dengan **Java Swing** melalui Apache NetBeans, menyediakan tampilan yang intuitif dan mudah digunakan.

## Teknologi yang Digunakan

* **Bahasa:** Java
* **IDE:** Apache NetBeans
* **Basis Data:** MySQL
